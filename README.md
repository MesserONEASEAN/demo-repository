# ONE ASEAN - Digital Unity Initiative

This repository hosts the ONE ASEAN website, promoting digital collaboration and unity across the Association of Southeast Asian Nations (ASEAN) member countries.

## About ONE ASEAN

ONE ASEAN represents our collective commitment to strengthening regional cooperation through digital transformation and technological innovation. This initiative aims to:

- Foster digital connectivity across all ASEAN member states
- Facilitate cross-border collaboration in technology and innovation
- Support sustainable development through digital solutions
- Strengthen regional economic integration

## Repository Contents

The repository includes:
- `index.html` - Main website showcasing ONE ASEAN objectives and member countries
- GitHub Actions workflows for automated deployment and validation
- Primer CSS styling framework for consistent design
- Responsive design supporting multiple devices and languages
